# Portfolio Pessoal

Este projeto é um portfolio pessoal criado com **React + Vite** para mostrar habilidades de programação e fluência em inglês.

## ✨ O que está incluído

- Apresentação profissional em landing page
- Seção de habilidades técnicas
- Destaque de projetos relevantes
- Seção dedicada à fluência em inglês técnico
- Contato profissional e resumo claro

## 🛠 Tecnologias utilizadas

- React
- Vite
- JavaScript (ES6+)
- CSS3

## ▶ Como rodar o projeto localmente

```bash
npm install
npm run dev
```
